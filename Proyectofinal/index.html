Proyecto final donde se utilizaron 3 sensores,un attiny85 y un OLED I2C

Diego Alonso Diaz Palacios
Ingenieria en Sistemas Computacionales
Fecha: noviembre 23,2023
Arduino Electrónica

- Resumen -

Este proyecto presenta el desarrollo de un wearable innovador utilizando un ATtiny85, un display OLED, y una serie de sensores como el MAX30105 y BMP085, enfocado en monitorear signos vitales como la frecuencia cardíaca y la saturación de oxígeno en sangre.

- Introducción -

El uso de la pantalla OLED es esencial para poder monitorear los estados o señales de sensores,botones,entre otro componentes,por lo que es de suma importancia saber utilizarlo.

La utilización del Attiny85 no solo representa una alternativa económica y compacta sino también una oportunidad para explorar y aprender nuevas configuraciones y métodos de programación, permitiendo a los desarrolladores expandir su conocimiento y habilidades en el vasto mundo de la electrónica digital.

- Materiales -

Los materiales utilizados incluyen un ATtiny85, un display OLED, sensor MAX30105, sensor BMP085, cables, resistencias, y una placa de desarrollo Arduino para la programación del ATtiny85.

- Desarrollo -

El desarrollo del proyecto comenzó con la programación del ATtiny85 usando un Arduino como programador. Posteriormente, se conectaron los sensores y el display OLED al ATtiny85, estableciendo un sistema capaz de monitorear y mostrar datos vitales.

El proyecto fue una oportunidad para explorar las capacidades del ATtiny85 en el manejo de múltiples sensores y la visualización de datos en tiempo real.

- Simulación -

  • Configuramos cada sensor individualmente en el espacio de trabajo.
  • Conecte los sensores a un microcontrolador, como Arduino, asegurandome de utilizar los pines adecuados, para el de impacto lo conectamos a un pin digial.
  • Fuimos probando uno por uno, con el sensor de impacto hubo algunas dificultades ya que se rompio el primero, pero funciono correctamente, con el sensor MAX30105 tuvimos algunas complicaciones ya que no marcaba los niveles en el OLED. Con el sensor BMP085 succedia lo mismo.

    • - Construcción -

  • Componentes: Nos aseguramos de tener todos los componentes necesarios: sensores, microcontrolador, resistencias, cables, placa protoboard y PCB(aunque no pudimos terminar correctamente este), y fuentes de alimentación.
  • Conexiones: Se realizaron las conexiones de cada sensor correctamente
  • Pruebas: Cuando se encendio el wearable hubo algunas fallas, el oximetro se sobreponia en la pantalla OLED junto con el sensor barometrico.
  • Ubicación: Se ubicaron correctamente los sensores para que estos no chocaran entre si.

    • - Resultados -

    Los resultados demostraron la eficacia del wearable en monitorear la frecuencia cardíaca y la saturación de oxígeno, destacando la precisión y la capacidad de visualización en tiempo real de los datos a través del OLED.
